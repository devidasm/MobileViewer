Detailed Description

Display a Dicom image, allowing the user to zoom in/out and pan.

The transforms specified in TransformsChain are applied to the image before it is displayed on the screen. The color space transform is applied automatically by the view so it doesn't have to be added to the TransformsChain.

The view renders the image asynchronously, by calculating the tiles to be displayed in a separate thread.
