Detailed Description

A data handler allows to read/write the data stored in a DataSet object without worrying about the data format.

Data handlers work on a local copy of the buffer so they don't need to worry about multithreading accesses.

Data handlers are also used to access to the decompressed image's pixels. Call Image::getDataHandler() to obtain a data handler.
