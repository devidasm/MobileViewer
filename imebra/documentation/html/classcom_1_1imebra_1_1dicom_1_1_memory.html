Detailed Description

This class holds an allocated array of bytes.

New instances of this class should be obtained through the class MemoryPool: call MemoryPool::getMemory() in order to obtain a new instance of memory.

long com.imebra.dicom.Memory.data ( byte[]  buffer)

Copies the bytes from the managed memory to the supplied array.

If the size of the bytes buffer is less than the memory's size, then the function just returns the required size, otherwise copies the content of the memory into the bytes buffer and then returns the number of bytes copied.
